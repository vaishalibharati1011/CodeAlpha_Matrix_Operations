# CodeAlpha_Matrix_Operations 
## Matrix Operations System developed in C Programming Language.

## Project Overview

The Matrix Operations System is a menu-driven C programming project developed as part of the CodeAlpha C Programming Internship.

This application performs various matrix operations using 2D arrays and functions. It allows users to perform matrix addition, matrix multiplication, matrix transpose, and display matrices through an interactive menu-driven interface.


## Features

- Matrix Addition
- Matrix Multiplication
- Transpose of Matrix A
- Transpose of Matrix B
- Display Matrix A and Matrix B
- Menu-Driven User Interface
- Function-Based Modular Programming
- Uses 2D Arrays


## Technologies Used

- C Programming Language
- Code::Blocks IDE
- GCC Compiler


## Concepts Used

- Functions
- Arrays
